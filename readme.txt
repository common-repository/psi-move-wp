Tool for replace URL (domain) in all database's tables.

== Description ==
We often working to build web site or web application on localhost until it finished.
Then we upload it to real server with domain/URL for it.
Sometimes we don't know some plugins or our application make new tables and save some data in serialize type in MySQL. Therefore this plugin can replace the old URL to the new one within all tables and serialize type. So no missing link anymore when you move your WordPress web site or application.

== Installation ==
1. Upload the psi-move-wp folder to the plugins directory in your WordPress installation.
2. Go to plugins list and activate "Moving WordPress".
3. To use it goto : Tools -> Moving WP.
